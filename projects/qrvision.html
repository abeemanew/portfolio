Generate QR Vision

Generator QR Invoice melalui enkripsi AES dan konversi Base64 untuk PT. TMMIN (QR Vision), serta pembuatan dan penyematan QR Code otomatis pada PDF Kwitansi PT. ADM (QR Delvi).

Tampilan Website

Zoom
Form Create Private Key

Form untuk memasukkan Private Key resmi yang diterbitkan oleh PT TMMIN.

Form Manual Input Invoice (PT. TMMIN)

Form untuk memasukkan data invoice secara manual, yang selanjutnya akan dienkripsi menggunakan AES-128-CBC dan digenerate menjadi QR Code.

Form Manual Input Kwitansi (PT. ADM)

Form untuk mengunggah PDF, mengisi data manual, serta menyematkan QR Code ke dalam dokumen.

Gunakan panah atau geser untuk melihat gambar lainnya

Tentang Project

Sistem ini dibangun menggunakan framework Django (Python) dan Django ORM (Object-Relational Mapper), dengan penyimpanan data pada PostgreSQL serta tampilan antarmuka yang dibangun menggunakan Bootstrap 5.

Fitur Utama

  • Autentikasi LDAP: Login terpusat menggunakan akun internal tanpa perlu registrasi ulang.
  • Generate QR Vision (TMMIN): Keamanan tingkat tinggi dengan enkripsi AES dan konversi data ke format Base64 sebelum proses generate QR.
  • Generate QR Delvi (ADM): Pembuatan otomatis dan penyematan (embedding) QR Code langsung ke dalam dokumen PDF kwitansi.
  • Sesi Keamanan: Fitur auto-logout otomatis untuk perlindungan akses akun secara menyeluruh.
